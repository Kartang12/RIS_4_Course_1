Нужна БД (делал на MS SQL Server)

В классе TreesService исзменить строку подключения и запросы в методах

UI
====================
(***)Все сам подгрузится при запуске, если сервер БД поднят. 
Если лежит, скорее всего будет ошибка, ибо лень делать проверки (не проверял).

Добавление
Ввести значения в 3 поля и нажать "Add"
"Type" выбрать из дропдауна 

Удаление
Выбрать строку в таблице
Кик по "Delete"

Редактирование
Клик по нужной строке
Ввести новые данные в 3 поля
Клик по "Edit"

"Load" подгружает данные из бд, но по причине (***) кнопка не нужна)))
