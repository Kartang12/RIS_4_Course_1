Клиент
==============
Сперва подгружаются данные о предметах и студентах
Клик по предмету- подгружается список занятий по этому предмету
Клик по занятию - подгрузится список присутствующих на этом занятии

Чтобы изменить отметку (присутствие/отсутствие), клик по "заголовку" СТРОКИ в табличке со студентами
Чтобы сохранить, клик по "Подтвердить"

"Добавить" добавит новое значение в список занятий по выбранному предмету
НЕ КЛИКАЙ по этому новому занятию, пока не нажмешь "Подтвердить" (изменять "оценки" в это время можно

"удалить" - удаляет только выбранное занятие (не предмет и не студента)

"Подключиться" кликать не надо по причине 1ой строки ^^^
